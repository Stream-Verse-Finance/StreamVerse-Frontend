# StreamVerse Hub

Interactive live gaming platform where viewers can directly influence the streamer's game through donations and events. Powered by Google Gemini AI and Web3 integration.

## Prerequisites

Before you begin, make sure you have the following installed on your machine:

*   **Node.js** (LTS version recommended)
*   **npm** (usually included with Node.js)
*   **Gemini API Key** (from Google AI Studio)

## Getting Started

Follow these steps to set up the project locally.

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/streamverse-hub.git
    cd streamverse-hub
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

## Running Development Server

To start the local development server and run the application:

1.  **Execute the following command:**

    ```bash
    npm run dev
    ```

2.  **Open your web browser and navigate to:** [http://localhost:5173/](http://localhost:5173/)

---

*Note: If port `5173` is already in use on your machine, check the terminal output for the actual port assigned by Vite (e.g., `5174`). Ensure your API_KEY is configured in your environment variables for AI features to work.*
